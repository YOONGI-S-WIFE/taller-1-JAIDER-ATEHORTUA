VOLVER A LA PAGINA PRINCIPAL

Tabla de eventos y ejemplos

Evento Ejemplo de aplicación
Click Abrir una ventana emergente al hacer clic en un botón.
Mouseover Cambiar el color de fondo de un elemento al pasar el cursor sobre él.
Submit Enviar un formulario cuando se hace clic en el botón "Enviar".
Keydown Mostrar un mensaje cuando una tecla se mantiene presionada.
Load Cargar contenido adicional en una página web cuando se carga completamente.
Scroll Realizar una acción cuando se realiza un desplazamiento vertical en una página.
Focus Realizar una acción cuando un elemento recibe el enfoque.
Blur Realizar una acción cuando un elemento pierde el enfoque.
Change Actualizar dinámicamente un campo de texto basado en la selección de un menú desplegable.
Resize Ajustar el diseño de una página cuando se cambia el tamaño de la ventana del navegador.
DoubleClick Ejecutar una función cuando se hace doble clic en un elemento.
abort interrumpir la carga de algo ya sea reiniciando o saliendose de la pagina
